# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of flutter_cloud_sync_webdav
- WebDAV protocol support with Basic Auth
- File upload, download, delete, and list operations
- Automatic directory creation
- Custom metadata storage as separate JSON files
- Compatible with Nextcloud, ownCloud, and other WebDAV servers
- Comprehensive error handling with typed exceptions
- Full documentation and setup guide
- Unit tests for provider configuration

## [0.1.0] - 2025-01-XX

### Added
- WebDAVProvider implementing CloudProvider interface
- WebDAVAuthService implementing CloudAuthService interface
- WebDAVStorageService implementing CloudStorageService interface
- Configuration validation for WebDAV servers
- README with setup instructions for Nextcloud and ownCloud
